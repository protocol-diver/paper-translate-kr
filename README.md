# paper-translate-kr
분산시스템, 통신 및 프로토콜에 관련된 RFC, 논문의 번역본을 보관하는 저장소입니다. <br>
한/영 변환 Github Page 운영에 대비하여 원본과 번역본 둘 다 저장합니다. 단, 페이지 관련 내용은 삭제하여 커밋합니다(번역하는 과정에서 원본의 페이징이 틀어질 것 같아 미리 제거). <br>
